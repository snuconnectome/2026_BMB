# 2026 뇌마음행동: 조교/교수용 실습 진행 가이드

## 실습 목표 세팅

이 수업에서의 AI 활용 철학은 명확합니다.
> "AI 자체를 맹목적으로 추종하거나 개발하는 것이 아니라, AI를 내 연구 질문(Research Question)을 탐구하기 위한 'Enabling Technology'로 적극 활용한다."

## 60분 실습 타임라인

### Phase 1. 워밍업 & 질문 발굴 (0~10분)

- 학생들에게 "뇌, 마음, 행동"에 관한 가장 흥미로운 질문 1개씩을 적게 합니다.
- 정답을 묻는 질문(What) 보다는, 원리(How/Why)나 가설을 묻는 질문이어야 합니다.

### Phase 2. 도구 연결 확인 (10~15분)

- Tailscale 및 Cline이 정상 연결되었는지 점검합니다.
- 학생들에게 터미널에 `System Check` 프롬프트를 보내보도록 지시합니다.

### Phase 3. 공통 파이프라인 경험: "로컬 에이전트 체감" (15~35분)

- `CLINE_PRACTICE_GUIDE.md`의 예제 프롬프트를 배포합니다.
- 조교는 돌아다니며 학생들이 에이전트의 Action을 Allow하고 있는지 점검합니다. ChatGPT와의 차이점(내 PC를 직접 제어한다는 것)을 인지시킵니다.

### Phase 4. 주도적 탐구 및 한계 봉착 경험 (35~50분)

- 학생이 워밍업에서 작성한 본인의 질문을 에이전트에게 던져 탐구하게 합니다.
- **핵심 포인트**: 학생이 "에이전트가 엉뚱한 소리를 하거나, 코드가 안 돌아간다"고 불평할 때, 조교는 직접 해결해주지 말고 **"그 문제를 에이전트에게 정확한 프롬프트로 설명하고 스스로 고치게 지시해봐"** 라고 유도합니다.

### Phase 5. 리뷰 및 성찰 (50~60분)

- 1~2명의 학생 사례를 공유합니다.
- 교수가 코멘트: "여러분은 이제 AI를 이용해 수십 편의 논문을 몇 분만에 정리했습니다. 중요한 것은 AI가 내놓은 답이 아니라, 여러분이 처음에 던진 **'질문의 수준'**입니다."

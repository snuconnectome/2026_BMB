# 2026 뇌마음행동: Cline 실습 가이드

**목표**: 스스로 발굴한 핵심 질문을 바탕으로, Agentic AI를 활용해 논문 검색, 데이터 요약, 코드(분석) 생성의 자동화 파이프라인을 경험하기.

---

## 1. 수업 실습 (60분)

여러분은 교재를 읽으며 생긴 '가장 궁금한 질문' 하나를 가지고 실습을 시작합니다.
예: "명상(안정 상태)이 뇌의 Default Mode Network 기능 연결성에 어떤 영향을 미칠까?"

### 실습 파이프라인 템플릿: "로컬에서 동작하는 AI 도우미 경험하기"

ChatGPT와 달리, Cline은 여러분의 PC에 폴더를 만들고, 논문을 다운받아 요약본을 저장합니다.
아래 프롬프트를 복사해 본인의 주제로 변경한 후 실행해보세요.

```text
나는 뇌마음행동 수업을 듣는 학생이야.
"명상이 Default Mode Network에 미치는 영향"에 대한 최신 연구가 궁금해.
다음 작업을 자동화해줘:
1. 프로젝트 루트에 'my_bmb_research' 폴더를 만들어.
2. 2024~2026년 출판된 관련 논문 10편을 검색해. (PubMed 또는 arXiv)
3. 논문 PDF 또는 초록 텍스트를 다운로드해서 'papers' 하위 폴더에 저장해.
4. 논문의 제목, 저자, 핵심 결론을 'metadata.csv'로 정리해.
5. 10편의 연구를 종합하여 내가 가진 질문에 대한 1페이지 분량의 마크다운 요약 보고서('summary_report.md')를 작성해.
```

## 2. 자유 실습 및 질문의 심화

1번 작업을 마친 뒤, 나온 요약 보고서를 읽어보고 **후속 질문**을 던지세요.
단순히 "알려줘"가 아니라, "**이 데이터를 어떻게 검증할지 파이썬 코드를 짜줘**" 와 같이 행동(Action)을 요청하세요.

- **예시**: "요약 보고서를 보니 A라는 개념과 B라는 개념이 상충해. A를 지지하는 fMRI 데이터 분석 파이프라인을 MNE-Python이나 Nilearn으로 짜줄 수 있어?"

## 3. 유의사항

- 에이전트가 제안하는 터미널 명령(설치, 디렉토리 생성 등)을 꼼꼼히 읽고 허용(Allow)하세요.
- 결과물에 오류가 있다면, 에러 메시지를 복사해서 "이 에러를 해결하고 다시 시도해" 라고 지시하며 협업 과정을 거치세요.

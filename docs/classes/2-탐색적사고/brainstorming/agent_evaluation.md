# Class 02 실습 아이디어 에이전트 다면 평가 (Agent Evaluation)

본 문서는 앞서 생성된 14개의 Raw 아이디어에 대해, 서로 다른 관점을 가진 3명의 가상 AI 에이전트 페르소나를 통과시켜 장단점을 분석하고 최종 상위 5개를 선별하기 위한 중간 평가 기록입니다.

## 👥 평가 에이전트(Persona) 위원회 구성

1. **Agent A (The Innovator - 창의성/흥미 전문가):** "이 아이디어가 학생들에게 얼마나 신선하고 폭발적인 도파민(흥미/효능감)을 줄 수 있는가?"를 평가합니다. 뻔한 문헌 조사를 혐오합니다.
2. **Agent B (The Pragmatic Engineer - 실현 가능성 전문가):** "비전공자 학생들이 단 30분 만에 코딩 없이 Cline/Cursor를 조작하여 에러 없이 결과물을 띄울 수 있는가?"를 가장 중요하게 봅니다. 복잡한 의존성이나 외부 API 병목을 우려합니다.
3. **Agent C (The Epistemologist - 교육 철학 전문가):** "단순히 예쁜 앱을 만드는 것을 넘어, 이 수업의 핵심인 'Action -> Prediction Error -> Steering' 철학을 경험할 수 있는가?"를 평가합니다.

---

## 🔍 에이전트 간섭 평가 (Top 5 후보군 요약 논의)

**▶ 후보 1. "나만의 뇌과학 매일 신문" (아이디어 1)**

* **Agent A (혁신):** 7/10. 결과물이 그럴싸하지만 조금 뻔한 감이 있음.
* **Agent B (실현당성):** 9/10. PubMed API 연동과 HTML 메일폼 생성은 Cline이 가장 잘하고 에러율이 적은 작업임. 30분 컷 완벽히 가능.
* **Agent C (교육):** 8/10. 정보의 능동적 큐레이션이라는 측면에서 'Intelligence Coordinator'의 기본기를 닦기에 아주 훌륭한 첫걸음.
* *총평:* 안정성과 기본기를 갖춘 훌륭한 선택지.

**▶ 후보 2. "과거와 현재의 대화" 타임머신 대시보드 (아이디어 2)**

* **Agent A (혁신):** 9/10. 패러다임 시프트를 시각적으로 한눈에 본다는 기획 자체가 학문적으로 매우 섹시함.
* **Agent B (실현당성):** 8/10. 화면 분할 UI와 텍스트 비교 컴포넌트는 React나 Vanilla JS로 금방 구현 가능함. 데이터베이스 구축 없이 하드코딩된 프롬프트 데이터로도 충분히 시뮬레이션 가능.
* **Agent C (교육):** 10/10. 지식이 어떻게 변화하고 논박당하는지(Prediction Error)를 보여주므로 수업 철학과 100% 일치함.
* *총평:* 학문적 깊이와 코딩 결과물의 멋짐을 동시에 잡은 베스트 후보.

**▶ 후보 3. 내 전두엽 상태 트래커 (아이디어 3)**

* **Agent A (혁신):** 10/10. '내 이야기'만큼 도파민이 터지는 주제는 없음. 학생들이 매일 쓰고 싶어 할 앱임.
* **Agent B (실현당성):** 7/10. 로직 자체는 단순하나, 뇌과학적 데이터를 퍼센티지로 치환하는 로직을 프롬프트로 잘 짜야 '그럴싸한' 결과가 나옴. UI 난이도는 무난.
* **Agent C (교육):** 9/10. 뇌/마음/행동을 타인의 이론이 아닌 내 몸과 삶의 물리적 데이터로 연결(Grounding)한다는 점에서 최고의 체화 학습.
* *총평:* 학생 참여도 1위 예상 후보. 필수 옵션으로 포함.

**▶ 후보 4. 착시 현상 발생기 및 원리 분석기 (아이디어 7)**

* **Agent A (혁신):** 9/10. 눈앞에서 착시가 그려지는 캔버스를 본드면 개발자/연구자 뽕(효능감)이 엄청날 것임. 시각적 임팩트 최상.
* **Agent B (실현당성):** 6/10. Canvas API를 Cline이 제어해야 하는데 미세한 픽셀 에러가 날 확률이 존재함. 하지만 성공하면 매우 멋짐.
* **Agent C (교육):** 8/10. 뇌가 '환각(Hallucination)'을 만들어내는 과정을 코드로 직접 구현해본다는 의미가 큼 (Buzsaki의 Inside-Out 철학).
* *총평:* 시각적 임팩트를 위해 준비해둘 만한 훌륭한 스페어 카드.

**▶ 후보 5. "내가 의사다" 텍스트 어드벤처 진단 게임 (아이디어 4)**

* **Agent A (혁신):** 8/10. 게임화(Gamification) 요소는 늘 재밌음. 챗봇을 환자 페르소나로 쓰는 발상의 전환.
* **Agent B (실현당성):** 9/10. 터미널 기반 파이썬 스크립트나 아주 단순한 챗 UI로 구현 가능하므로 Cline 구현 난이도 최하. 에러 발생 확률 거의 0.
* **Agent C (교육):** 9/10. 정답을 묻는 게 아니라, 내가 질문(사전지식/가설)을 던지고 환자의 대답(예측오류)을 바탕으로 가설을 업데이트하는 완벽한 'Active Inference' 훈련장.
* *총평:* 구현은 가장 쉽지만 교육적 철학은 가장 깊게 담긴 역설적인 최고의 카드.

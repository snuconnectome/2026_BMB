# Class 02 실습 아이디어 뱅크 (Raw Ideas)

사용자의 요청에 따라, 코딩 없이 Agentic AI(Cline)를 활용해 '뇌-마음-행동' 과목의 효능감을 극대화할 수 있는 창의적인 실습 아이디어들을 최대한 많이 생성합니다.

1. **"나만의 뇌과학 매일 신문" (The Neuro-Daily)**
    * 특정 키워드(예: 수면, 우울증)의 최신 논문을 PubMed에서 매주 긁어와 웹진 형태의 뉴스레터로 자동 발행하는 스크립트 작성.
2. **"과거와 현재의 대화" 타임머신 대시보드 (Paradigm Explorer)**
    * 90~00년대 핵심 논문(예: 거울 뉴런)과 2024~25년 트렌드 논문을 나란히 비교하며 패러다임 변화를 시각화하는 인터랙티브 웹 앱.
3. **내 전두엽 상태 트래커 (My Inner-State Tracker)**
    * 매일의 수면/스트레스/카페인 데이터를 입력하면 뇌과학 지식을 바탕으로 현재 내 뇌의 퍼포먼스를 예측해주는 모바일 웹 앱.
4. **"내가 의사다" 텍스트 어드벤처 진단 게임 (Virtual Clinic)**
    * 조현병, ADHD 등 특정 질환의 환자 페르소나와 대화하며 증상을 유도하고 진단하는 터미널/웹 기반의 텍스트 추리 게임.
5. **사이드 이펙트 시뮬레이터 (Drug Side-Effect Visualizer)**
    * 특정 신경정신과 약물(예: SSRI, 도파민 작용제)의 작용 기전과 부작용 확률을 시각적인 차트나 인포그래픽 모듈로 렌더링해주는 웹 앱.
6. **"내 두뇌의 MBTI" - 신경전달물질 기반 성격 테스트 제작**
    * MBTI 대신, 도파민/세로토닌/테스토스테론/에스트로겐 4가지 척도를 기반으로 헬렌 피셔(Helen Fisher) 모델 등을 차용한 나만의 심리테스트 웹사이트 만들기.
7. **착시 현상(Optical Illusion) 발생기 및 원리 분석기**
    * Cline에게 코딩을 시켜 유명한 뇌과학적 착시 현상(그리드 착시, 색상 동화 등)을 캔버스(Canvas) 위에 직접 렌더링 화면으로 구현하고, 그 기전을 옆에 띄우는 앱.
8. **알파고 쇼크 - '나의 감정' 워드 클라우드 제너레이터**
    * 오리엔테이션 과제로 작성했던 학생들의 감상문 텍스트를 입력받아, 빈도수와 감각 단어를 분석하여 아름다운 '감정 워드클라우드' 이미지로 변환해주는 스크립트.
9. **"거짓말 탐지기" 프로토타입 앱 (fMRI 콘셉트)**
    * 사용자의 타이핑 속도나 마우스 움직임 패턴 데이터를 뇌과학적 집중도나 인지 부하(Cognitive Load) 개념과 끼워맞춰, 유머러스하게 '현재 거짓말을 하고 있는가'를 판별하는 장난감 웹.
10. **뇌 구조 3D 탐험가 (Three.js 뇌 모델링)**
    * Three.js 라이브러리를 활용해 터미널 명령어 하나로 웹 브라우저에 3D 뇌를 띄우고, 마우스오버 시 각 부위(편도체, 전두엽 등)의 행동적 기능을 설명해주는 심플한 3D 시각화 앱.
11. **"도파민 디톡스" 뽀모도로 타이머**
    * 일반적인 타이머가 아니라, 휴식 시간에 스마트폰을 보면 알람이 초기화되고, 작업 시간에 도파민 분비 기전을 유머러스한 문구로 응원해주는 뇌과학 기반 맞춤형 타이머 앱.
12. **나만의 AI 연구 조교 "라부아지에" (논문 Q&A 챗봇 UI)**
    * 로컬에 다운로드 받은 PDF 논문 1편을 업로드하면, 그 논문에 대해서만 묻고 답할 수 있는 심플한 챗봇 UI(Gradio 또는 Streamlit)를 10분 만에 자동 생성하기.
13. **"수면의 건축학" 수면 주기(Sleep Cycle) 시각화 도구**
    * 가상의 수면 뇌파 데이터(REM, NREM 등)를 엑셀이나 CSV 형태로 주고, 이를 예쁜 D3.js 그래프로 시각화하여 내 수면의 질을 분석하는 대시보드 구축.
14. **음악이 뇌에 미치는 영향 분석기 (Spotify API 연동)**
    * 자신이 자주 듣는 스포티파이 플레이리스트의 분위기 데이터를 API로 가져와서, "이 음악은 플레이어의 도파민을 X%, 옥시토신을 Y% 자극함"을 시각적으로 보여주는 재미있는 웹.

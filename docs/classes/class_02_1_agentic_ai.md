# Class 2: Agentic AI를 이용한 뇌-마음-행동의 학습

**일시:** 2026년 3월 5일 (목)
**강연자:** 차지욱 교수 (심리학과)

## 🎯 수업 목표

1. **Agentic AI의 개념 이해**: 수동적인 지식 검색 도구(전통적 검색엔진, 단순 챗봇)와 능동적인 문제 해결 파트너(Agentic AI)의 차이점 이해.
2. **NotebookLM 온보딩 (실습)**: 본 강의의 주된 지식 베이스이자 AI 비서인 구글 NotebookLM 사용법 익히기.
3. **프롬프트 엔지니어링 (Prompt Engineering) 기초**: "좋은 질문"이란 무엇인가? 뇌과학/심리학 논문을 분석할 때 AI에게 어떻게 질문을 던져야 하는가 학습.
4. **Prompt Audit 개념 소개**: 과제 수행 시 AI와의 상호작용 내역을 스스로 평가하고 제출하는 방법 실습.

## 🕒 진행 순서 (예상 시간: 75분)

### 1. 도입: 지식을 외우는 시대의 종말 (15분)

- 정보의 과잉 시대에 필요한 능력: 파편화된 지식을 암기하는 것이 아니라, 데이터를 비판적으로 엮고 새로운 가설을 세우는 능력.
- **LLM vs Agentic AI**: 단답형 기계가 아닌, "목표(Goal)"를 주고 도구(Tools)를 활용해 "계획(Planning/Execution)"을 세워 달성해가는 에이전트 개념 소개.
- 2026 BMB 코스 내내 여러분과 함께할 가상의 AI 동료 연구자(CoScientist) 소개.

### 2. Agentic AI (Cline) 워크샵 - 💻 심층 문헌 조사 실습 (30분)

- **사전 준비 점검**: Cursor 및 Cline 익스텐션 셋업 확인 (오리엔테이션 권장 사항).
- **실습 목표**: 오리엔테이션 과제로 준비해 온 'Deep Question'을 Agentic AI를 통해 파고들기.
- **Activity: 나만의 문헌 요약 에이전트 만들기**
  1. Cursor의 Cline에게 자신이 고른 첫 번째 Deep Question과 관련된 핵심 주제어를 줍니다.
  2. Cline에게 해당 주제와 관련된 가장 최신/중요 논문 5편을 검색, 다운로드하고 요약하게 시킵니다.
  3. **프롬프트 예시:**
     > "나는 뇌마음행동 기초 강의를 듣는 학생이야. '작업기억(Working Memory)과 전전두엽'에 대해 막 관심이 생겼어. PubMed에서 이 주제와 관련된 최신 논문 5편을 찾고, 내 로컬 프로젝트 폴더 안에 `workspace` 폴더를 만들어서 초록(Abstract)을 텍스트 파일로 저장한 다음, 각각이 지지하는 주장을 1페이지 마크다운 파일로 비교 요약해 줘."
- **AI와의 줄다리기(Steering)**: Cline이 잘못된 논문을 가져오거나 요약 방향이 틀렸을 때, 어떻게 궤도를 수정(Steering)하는지 교수님 데모 시연 및 학생 실습.

### 3. NotebookLM 및 Prompt Audit - 💻 지식의 구조화 (25분)

- **지식의 하우스(Brain) 짓기**: Cline이 요약해준 마크다운 텍스트와 원문 PDF들을 구글 NotebookLM에 업로드.
- **주요 기능 실습**:
  - Source Guide를 통한 전체 맥락 파악.
  - 특정 개념 "Grounding"하여 질문하기 (문서에 팩트 기반하여 대답하게 유도하기).
- **프롬프트 오딧 (Prompt Audit)**:
  - 수업 과제 제출 시, 단순히 AI가 적어준 답을 복붙하는 것은 0점.
  - 자신이 Cline이나 NotebookLM에 어떤 질문을 던졌고, AI의 답변 중 어떤 부분이 유용했거나 예측 오류(Prediction Error)를 발생시켰는지 캡처/기록하여 '거인의 어깨 오딧 로그' 형태로 제출하는 방법 안내.

### 4. 실습 과제 안내 및 Q&A (10분)

- **주말 과제 (첫 eTL 제출 과제)**:
  - 오늘 실습한 Cline 문헌조사와 NotebookLM 대화 내용을 바탕으로 가장 깊이 있는 깨달음을 준 "초기 프롬프트 -> AI의 예측 오류 유발 답변 -> 모델이 수정된 발전된 프롬프트" 세트 1개를 첫 번째 '오딧 로그' 포맷으로 eTL에 공유하기.
